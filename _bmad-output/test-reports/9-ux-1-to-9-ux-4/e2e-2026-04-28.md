## Bug Found & Fixed

### Bug #1 — `data-slot="agent-lane"` missing from `AgentLane` component

**File:** `nowing_web/components/new-chat/orchestra/agent-lane.tsx:125`

**Impact:** Playwright tests `whale-tracker.spec.ts` T14/T15 use `[data-slot="agent-lane"]` selector — would return 0 results and fail all count assertions.

**Root cause:** `AgentLane` component used `data-agent-id` and `data-status` but no `data-slot`.

**Fix applied:**
```tsx
// Before
data-agent-id={agentId}
data-status={status}

// After
data-slot="agent-lane"
data-agent-id={agentId}
data-status={status}
```

**Verified:** `document.querySelectorAll('[data-slot="agent-lane"]')` → 6 results on live thread 46 ✅
